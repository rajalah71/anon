# anon

A package containing tools for data anonymization for RStudio.

## Installation

1. Install "devtools" package on R
    - install.packages("devtools")
2. Install the "anon" package on R
    - devtools::install_github("rajalah71/anon")
3. Load the "anon" library
    - library(anon)

## Usage

The package contains (2 at the moment) functions to anonymize your data, as well as helper functions to modify different columns in different ways.
